Creare un progetto

Nuovo progetto da zero, da template, o importando direttamente da GitHub.

Progetto vuoto

Un progetto vuoto ti dà un ambiente Linux completo con un terminale bash/zsh pronto all'uso. Da qui puoi installare qualsiasi cosa: framework, librerie, tool. È il tuo server personale nel cloud.

Setup manuale

$ mkdir my-app && cd my-app
$ npm init -y
$ touch index.js

Quando sceglierlo

Ideale se vuoi il controllo totale sulla configurazione, oppure se lavori con un framework non incluso nei template.

Template

I template sono progetti pre-configurati pronti all'uso. Al momento supportiamo:

Template disponibili

  • React — Create React App con hot reload
  • Node.js — Server Express minimale
  • Python — Flask app con struttura base
  • HTML/CSS/JS — Sito statico con live preview
  • Next.js — Full-stack React framework

Personalizzazione

I template sono punti di partenza: puoi modificare, aggiungere ed eliminare qualsiasi file dopo la creazione del progetto.

Import da GitHub

Collega il tuo account GitHub (vedi Collegare GitHub) e clona qualsiasi repository direttamente nell'app. Supportiamo repo pubbliche e private.

Repo pubbliche e private

Per le repo pubbliche non serve autorizzazione. Per le private, collega il tuo account GitHub tramite OAuth.

Nota: Ogni progetto ha il suo ambiente isolato. Le dipendenze installate in un progetto non interferiscono con gli altri.
