Debug con AI

Invia errori all'AI Agent e ottieni fix automatici e spiegazioni dettagliate.

Inviare un errore

Quando incontri un errore nel terminale o nell'editor, puoi:

Copia e incolla

Copia l'errore dal terminale e incollalo nella chat AI. Includi anche il contesto per risultati migliori.

Pulsante "Fix"

Tocca il pulsante "Fix" che appare automaticamente sugli errori nel terminale — l'errore viene inviato all'AI con un solo tap.

Seleziona codice

Seleziona codice nell'editor e chiedere all'AI di analizzarlo direttamente dal contesto.

Fix automatici

L'AI Agent può applicare fix direttamente ai file del tuo progetto. Quando l'AI suggerisce una modifica, vedrai un pulsante "Applica" che sovrascrive il codice nel file corretto.

Applicare le modifiche

Il pulsante "Applica" inserisce il fix nel file giusto. Puoi rivedere le modifiche prima di salvarle definitivamente.

Annullare un fix

Se il fix non è corretto, puoi annullare le modifiche. Il codice originale viene preservato fino alla conferma.

Spiegazioni del codice

Seleziona un blocco di codice e chiedi all'AI "Spiega questo codice". L'AI analizzerà la logica, spiegherà cosa fa ogni parte e suggerirà eventuali miglioramenti.

Analisi della logica

L'AI scompone il codice riga per riga, spiegando il flusso logico e identificando potenziali problemi.

Suggerimenti di miglioramento

Oltre alla spiegazione, l'AI può suggerire ottimizzazioni, best practice e pattern migliori per il tuo codice.

Suggerimento: Quando incolli un errore, includi anche il contesto (file, funzione, cosa stavi cercando di fare). Più contesto dai, più preciso sarà il fix.
