Richieste AI

Come funzionano i limiti delle richieste AI e cosa succede quando finiscono.

Cosa conta come richiesta

Ogni messaggio che invii all'AI Agent conta come una richiesta. Questo include:

Conta come richiesta

  • Chiedere di generare codice
  • Chiedere spiegazioni
  • Chiedere fix per errori
  • Qualsiasi interazione con la chat AI

Non conta come richiesta

Usare l'editor, il terminale, la preview live, GitHub — tutto il resto funziona senza limiti.

Limiti per piano

Confronto limiti

  • Free: 100 richieste/mese
  • Plus: 2.500 richieste/mese
  • Pro: 10.000 richieste/mese
  • Business: Illimitate

Rinnovo

Le richieste si rinnovano ogni mese dalla data di iscrizione. Le richieste non utilizzate non si accumulano.

Data di rinnovo

Il contatore si resetta alla data esatta della tua iscrizione, non il primo del mese.

Quando finiscono

Se esaurisci le richieste mensili:

Cosa continua a funzionare

  • L'editor, il terminale e GitHub continuano a funzionare senza limiti
  • Solo l'AI Agent viene messo in pausa

Come sbloccare

Puoi fare upgrade al piano superiore per sbloccare subito altre richieste, oppure aspettare il rinnovo mensile.

Nota: Puoi controllare quante richieste ti restano direttamente nell'app, nelle impostazioni del piano.
