Preview live

Vedi il risultato del tuo codice in tempo reale, direttamente dall'app.

Come funziona

Quando avvii un server web nel terminale (es. npm start, python -m http.server), Drape rileva automaticamente la porta e ti offre di aprire la preview live.

Rilevamento automatico

Drape monitora le porte attive e mostra un avviso quando un server viene avviato. Tocca per aprire la preview.

Browser integrato

La preview mostra il tuo sito/app in un browser integrato che si aggiorna automaticamente a ogni modifica salvata.

Hot reload

Per i framework che lo supportano (React, Next.js, Vite...), il hot reload funziona automaticamente. Modifica un componente, salva, e vedi il cambiamento istantaneamente senza ricaricare la pagina.

Framework supportati

React, Next.js, Vite, Vue CLI e qualsiasi framework con supporto HMR (Hot Module Replacement) nativo.

Porte supportate

Drape supporta qualsiasi porta tra 1000 e 65535. Le porte più comuni vengono rilevate automaticamente:

Porte comuni

  • 3000 — React, Next.js, Express
  • 5173 — Vite
  • 8000 — Python, Django
  • 8080 — Vue, generici

Porte personalizzate

Se usi una porta non standard, puoi configurarla manualmente nelle impostazioni della preview.

Nota: La preview è accessibile solo da te. Il tuo progetto non è esposto a internet fino a quando non fai deploy esplicitamente.
