Changelog

Tutte le novità e gli aggiornamenti di Drape.

v1.2 Febbraio 2026

5 modelli AI, nuovo editor

  • Nuovo Supporto Claude 4.6 Opus, GPT 5.2, Gemini 3.0 Pro e Gemini 3.0 Flash — ora 5 modelli AI disponibili
  • Nuovo Selettore modello inline — cambia modello AI con un tap durante la conversazione
  • Miglioria Editor con syntax highlighting potenziato per 20+ linguaggi
  • Miglioria Performance AI Agent migliorata del 40% — risposte più veloci e precise
  • Fix Stabilità terminale migliorata su progetti con molti file
  • Fix Risolti crash occasionali all'apertura di file molto grandi
v1.1 Gennaio 2026

GitHub Integration

  • Nuovo Importa repository da GitHub — clona qualsiasi repo con un tap
  • Nuovo Push, commit, pull direttamente dal telefono
  • Nuovo Gestione branch — crea, switcha e mergia branch
  • Miglioria Preview live con hot reload — le modifiche si vedono in tempo reale
  • Miglioria Nuovo design della barra di navigazione file
  • Fix Risolti problemi di connessione su reti mobili instabili
v1.0 Dicembre 2025

Lancio su App Store

  • Nuovo Lancio ufficiale su App Store — il primo IDE mobile completo
  • Nuovo Terminal bash/zsh completo con supporto a tutti i comandi standard
  • Nuovo AI Agent autonomo con Claude e GPT — descrivi e costruisci
  • Nuovo File explorer con navigazione cartelle e creazione file
  • Nuovo Code editor con syntax highlighting e autocompletamento
  • Nuovo Preview live per web app — testa il risultato senza deploy
