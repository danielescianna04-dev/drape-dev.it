FAQ

Domande frequenti

Posso usare Drape per progetti veri? +

Assolutamente sì. Drape è un IDE completo con terminale reale, non una simulazione. Puoi eseguire npm, git, python, installare dipendenze, testare API e fare push su GitHub. È pensato per sviluppatori che vogliono lavorare su progetti reali quando non hanno il laptop.

Come si confronta con VS Code o Cursor? +

Drape non sostituisce il tuo IDE desktop — lo estende. È l'unico IDE mobile completo con AI integrata, terminale reale e GitHub. Quando sei in treno, in pausa pranzo o sul divano, Drape ti permette di continuare a lavorare come faresti dal PC.

Come funziona il terminal su mobile? +

Il terminal di Drape si connette a container Docker isolati nel cloud. Ogni progetto ha il suo ambiente con accesso completo a bash/zsh, npm, git e tutti i tool di sviluppo. La latenza è minima e l'esperienza è identica a un terminale desktop.

Quali modelli AI sono disponibili? +

Il piano Starter include Claude 4.5 Sonnet e Gemini 3 Flash. I piani Go (€22.99/mese) e Pro (€39.99/mese) sbloccano anche Claude 4.5 Opus, GPT-5.2, Gemini 3 Pro e Llama 3.3 70B. Puoi cambiare modello in qualsiasi momento.

Posso importare le mie repository esistenti? +

Sì, è una delle funzionalità principali. Clona qualsiasi repository da GitHub, lavora con l'AI, avvia la preview live per testare le modifiche e pusha direttamente dall'app. Commit, pull, branch — tutto supportato.

Cosa succede al mio codice se cancello l'abbonamento? +

Il codice è tuo, sempre. Fai push su GitHub prima di cancellare e avrai tutto. Drape non tiene in ostaggio il tuo lavoro — è un tool, non una gabbia.

Come funziona la fatturazione? +

Tutti i pagamenti sono gestiti tramite Apple In-App Purchase. L'abbonamento si rinnova automaticamente e puoi cancellarlo in qualsiasi momento dalle impostazioni del tuo Apple ID.

I miei dati e il codice sono al sicuro? +

Assolutamente. Ogni progetto gira in un container Docker isolato. I dati sono crittografati in transito (TLS) e a riposo. Non vendiamo né condividiamo i tuoi dati. Leggi la nostra Privacy Policy per i dettagli.
