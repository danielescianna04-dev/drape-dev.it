Features

01

Autonomous AI Agent

Describe what you want and AI creates, edits and organizes files for you. Multi-file editing, dependency installation, command execution — all with a single prompt.

02

Full Terminal

Real Bash and Zsh in isolated Docker containers. Run npm, git, python and any command just like on a desktop terminal.

03

5 AI Models

Claude 4.5 Sonnet, Claude 4.6 Opus, GPT 5.2, Gemini 3.0 Pro and Gemini 3.0 Flash. Switch models anytime during development.

Clone any repo from GitHub, GitLab or Bitbucket and work on it with AI. Start the preview, test changes live and push — just like on desktop.

File Explorer & Editor

Navigate your project, create and edit files with syntax highlighting. Full cloud filesystem management with undo and redo.

FAQ

Frequently Asked Questions

How it works il terminal su mobile? +

Drape's terminal connects to isolated Docker containers in the cloud. Each project has its own environment with full access to bash/zsh, npm, git and all development tools. Latency is minimal and the experience is identical to a desktop terminal.

Which AI models are available? +

The Starter plan includes Claude 4.5 Sonnet and Gemini 3.0 Flash. Go (€22.99/mo) and Pro (€39.99/mo) plans also unlock Claude 4.6 Opus, GPT 5.2 and Gemini 3.0 Pro. You can switch models at any time.

Can I import my existing repositories? +

Yes, it's one of the main features. Clone any repository from GitHub, GitLab or Bitbucket, work with AI, start the live preview to test changes and push directly from the app. Commit, pull, branch — all supported.

How it works la fatturazione? +

All payments are handled through Apple In-App Purchase. The subscription renews automatically and you can cancel anytime from your Apple ID settings.

Are my data and code safe? +

Absolutely. Each project runs in an isolated Docker container. Data is encrypted in transit (TLS) and at rest. We never sell or share your data. Read our Privacy Policy for details.
