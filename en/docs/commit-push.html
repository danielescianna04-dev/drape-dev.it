Commit & push

Save your changes and push to GitHub directly from Drape.

Make a commit

Stage and commit

$ git add .
$ git commit -m "Add login feature"

Git panel

The Git panel shows modified files with a visual diff. Select the files to include, write a commit message and confirm.

Push to remote

Push command

$ git push origin main

Push via UI

Use the "Push" button in the Git panel. Drape handles authentication automatically using your connected GitHub account.

Pull changes

Download updates

$ git pull origin main

Pull the latest changes from the remote repository to stay in sync with your team.

Branches

Create and switch

$ git checkout -b feature/new-ui
$ git branch -a

Merge

$ git merge main

Note: All standard git commands work in the Drape terminal. The Git panel is a visual shortcut, but the terminal gives you full control.
