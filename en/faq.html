Frequently Asked Questions

Everything you want to know about Drape, all in one place.

General
Drape is a complete mobile IDE with AI. It lets you code directly from your iPhone with a real bash/zsh terminal, 8 AI models, GitHub integration, code editor and live preview. It's built for real developers who want to work even when they don't have their laptop.
Drape is available on iPhone via the App Store. The Android version is in development. You can request early access for Android.
Yes! The Free plan includes 100 AI requests per month, full editor, terminal and GitHub access. You can upgrade to Plus, Pro or Business for more AI requests and advanced features. See all plans.
The code runs on secure cloud servers. Your phone acts as the control interface — you write code, run commands and see results in real time. This means you have all the power of a server, without hardware limitations of the phone.
Features
Drape supports 5 AI models: Claude 4.5 Sonnet, Claude 4.6 Opus, GPT 5.2, Gemini 3.0 Pro, and Gemini 3.0 Flash. You can switch models at any time based on the type of task you're working on.
Absolutely. Drape's terminal is a full bash/zsh terminal. You can use npm, yarn, pip, cargo, and any command-line tool. Everything works just like on your Mac or PC.
You connect your GitHub account directly from the app. Then you can clone any repository (public or private), edit code, create commits and push — all from your phone. We also support branches, pull and merge.
Yes. For web projects, Drape offers a live preview that updates automatically as you edit your code. You can see the result in real time without having to deploy.
Pricing & Plans
Free: free forever. Plus: $20/month (2,500 AI requests). Pro: $50/month (10,000 requests). Business: $200/month (unlimited requests). Compare plans.
Yes, you can upgrade or downgrade at any time from the app settings. Changes take effect from the next billing cycle.
You can keep using the editor, terminal and GitHub without limits. Only AI features are paused until your monthly renewal. You can also upgrade to the next plan for additional requests right away.
Technical
Yes, Drape requires an internet connection as the code runs on cloud servers. It works well even on mobile connections (4G/5G). Sessions are optimized to reduce data usage.
Absolutely. Each user has an isolated environment. Code is encrypted in transit and at rest. We never access your code and don't use it for AI training. GitHub connection uses secure OAuth.
All of them. Drape provides a full Linux environment in the cloud, so it supports any language: JavaScript/TypeScript, Python, Rust, Go, Java, Ruby, PHP, C/C++, Swift and many more. If it runs on Linux, it runs on Drape.
